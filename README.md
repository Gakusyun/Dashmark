# DashMark

**Links, organized. Instantly.** 

**链接有序，即刻可达。**

DashMark 是一个专注于分组书签的极简起始页应用。它安静、迅速、不打扰，只在你需要时，将你带到正确的地方。

## ✨ 特性

- 📁 **分组管理书签**
  - 单层分组结构，清晰直观
  - 一个链接可同时属于多个分组
- 🔍 **快速搜索**
  - 支持 Google / Bing / 百度
  - 仅在确认操作时跳转，不做多余行为
- 🌗 **深色 / 浅色主题**
  - 支持跟随系统或手动切换
- 💾 **数据备份与导入**
  - 本地数据存储（localStorage）
  - 支持 `.json` 与 `.json.gz`（gzip 压缩）格式
- 🎨 **Material UI 设计**
  - 基于 MUI 组件与设计规范
  - 简洁、一致、响应式
